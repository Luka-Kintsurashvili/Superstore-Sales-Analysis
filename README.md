# Superstore Sales & Profit Analysis

## Project Overview
This project performs an Exploratory Data Analysis (EDA) on the Sample Superstore dataset using Python. The goal is to identify key sales drivers and seasonal profit trends to assist in business decision-making.

## Key Insights
* **Category Performance:** Technology leads in total sales, while Office Supplies show the highest transaction frequency.
* **Profitability:** Identified a consistent profit surge in Q4, likely driven by holiday demand.
* **Top Products:** Canon imageCLASS Copiers were identified as the highest-profit individual product.

## Tech Stack
* **Language:** Python 3.13
* **Libraries:** Pandas (Data Manipulation), Matplotlib & Seaborn (Visualization)
* **Environment:** VS Code / Jupyter Notebooks
